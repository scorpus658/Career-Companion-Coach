
# Epic: Position Yourself

[[B. Value chain with objectives]]

---

## Objective

Make the user's LinkedIn profile clearly signal fit for their target roles.

## JTBD

When a recruiter looks me up, I want my profile to reflect my strengths and intent, so I get shortlisted instead of skipped.

---

## User Stories

### 1. LinkedIn Profile Optimization

As a job seeker, I want to receive specific, actionable suggestions to improve my LinkedIn profile based on my target roles, so that I pass recruiter filters and clearly signal my fit.

Acceptance Criteria:
- Suggestions are specific to the user's target role and industry, not generic
- Each suggestion explains why it improves fit for the target role
- User can mark suggestions as applied and see a profile completeness score update accordingly
